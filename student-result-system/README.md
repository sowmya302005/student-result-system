# Student Result Management System

A simple PHP and MySQL based web application to manage student results.

## Features
- Add student details
- View student records
- Update student information
- Delete student records
- Bootstrap-based responsive UI

## Tech Stack
- PHP
- MySQL
- XAMPP
- HTML, CSS, Bootstrap

## How to Run
1. Install XAMPP
2. Copy project to `htdocs`
3. Import the SQL file into phpMyAdmin
4. Start Apache & MySQL
5. Open `http://localhost/student-result-system`
